#Comandos basicos de la consola linux

ls -> lista los archivos de la ruta actual (posici칩n) actual
cd [directorio] -> cambia de directorio
mkdir [directorio] -> crea un directorio 
touch [ archivo]-> crea un archivo
cat [ archivo] muestra el contenido de un archivo (sin abrirlo) 
nano [archivo] -> abre el archvo con el editot de texto nano 
rm [archivo] -> elimina el archivo
cd .. -> vuelve al directorio anterior
rm -r directorio -> borra recursivamente todos los archivos dentro del directorio
